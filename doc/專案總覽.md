## 一、專案目標

**願景**：成為台灣社區鄰里之間最被信任的工具共享平台。
*(來源：《商業企劃書》§10、《會議記錄_專案啟動會議》二.1、《專案簡報》結語)*

**首年 KPI（上線一年內）** — 各文件一致：

| 指標 | 目標 |
| --- | --- |
| 累積下載數 | 10,000 |
| 月活躍用戶（MAU） | 3,000 |
| 每月成功借用 | 500 筆 |
| Premium 付費轉換率 | 8%（約 240 位付費會員） |
| 首年營收估計 | 約 35～40 萬元（市場驗證期，重點在用戶成長） |
| 損益兩平 | 預估上線後第 2 年第 3 季 |

*(來源：《商業企劃書》§7、《會議記錄_專案啟動會議》二.1、《專案簡報》§08；營收／損益估計僅見於《商業企劃書》)*

## 三、專案時程 ⚠️ 有版本變動

**最新有效版本（以《會議記錄_第二次進度會議》2026-04-15 為準）：**

| 里程碑 | 階段 | 原訂期間 | 最新期間 |
| --- | --- | --- | --- |
| M1 | 需求確認與設計 | 2026-01 ~ 02 | 2026-01 ~ 02（未變） |
| M2 | MVP 開發 | 2026-03 ~ 05 | **順延至 2026-06 中** |

*(原訂：《商業企劃書》§9、《專案簡報》§06、《會議記錄_專案啟動會議》二.3；最新：《會議記錄_第二次進度會議》R-05、R-07，並由《開發進度與技術決策紀錄》§4 佐證)*
