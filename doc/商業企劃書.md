# ShareBox 社區共享工具平台

# 商 業 企 劃 書

### 版本 v1.0 ｜ 2026 年 1 月

### 提案單位：ShareBox 專案團隊

## 1. 執行摘要

ShareBox 是一款以社區為單位的工具共享行動 App。家用工具購買後使用頻率極低、佔據收納空
間，而鄰居之間想互借卻缺乏信任與保障機制。ShareBox 透過地圖媒合、押金代管與評價制度，讓
同社區住戶能安全地互借閒置工具。本企劃書說明市場機會、競爭分析、商業模式與營運目標。

## 2. 市場痛點與需求

- 家用工具（電鑽、梯子、吸塵器、露營裝備等）一年使用常不到 3 次。
- 工具佔據居家收納空間，造成資源浪費。
- 鄰居間想互借，但缺乏押金保障與損壞賠償的媒介。
- 既有租借店租金高、據點少、品項受限，且需親自往返。

## 3. 市場規模與趨勢

- 全台住宅約 900 萬戶，其中都會區公寓大廈住戶估計約 350

萬戶，為初期可觸及市場（TAM）。
- 共享經濟、ESG 永續消費、減塑減廢意識上升，推升閒置資源共享需求。
- 首階段鎖定雙北、桃園、台中的新型社區大樓。

## 4. 競品分析

| 競爭者 | 類型 | 主要限制 |
|---|---|---|
| Facebook 社團／LINE 群組 | 間接競品 | 免費但無金流、無保障、媒合效率低 |
| 連鎖工具租借店 | 間接競品 | 品項固定、據點少、需親自往返 |
| 量販店租借服務 | 間接競品 | 僅限特定品項 |

目前市場尚無成熟的「社區工具共享 App」直接競品。ShareBox
的差異化在於：在地化（同社區、步行可達）、信任機制（實名 + 評價 +
押金代管）、低成本（活用鄰居閒置資源）。

## 5. 產品與核心功能

ShareBox 第一版（MVP）提供：工具刊登、地圖搜尋附近工具、借用申請與審核、訂單即時聊天
、押金代管金流、歸還與評價、推播通知。信用評分與平台保險規劃於後續版本推出。

## 6. 商業模式與營收來源

| 營收來源 | 說明 |
|---|---|
| 免費版 | 基本刊登與借用功能，免費使用 |
| Premium 訂閱 | 每月 99 元（年繳 990 元），提供無限刊登、優先曝光、進階保障額度 |
| 交易服務費 | 每筆成功付費借用，平台抽取 10% 服務費 |

| 保險手續費 | 押金代管與保險方案之手續費收入 |
|---|---|
| 未來：B2B 與廣告 | 社區管委會方案、在地五金行廣告 |

## 10. 風險與結語

主要風險為金流串接、保險方案法務遵循與用戶成長速度。ShareBox 切入尚無直接競品的社區工具
共享市場，以信任機制為核心競爭力，期能成為台灣社區鄰里共享經濟的代表平台。
