## 一、會議目的

正式啟動 ShareBox 社區共享工具平台專案，確認專案目標、範圍、時程、預算與團隊分工。

## 二、討論事項

### 1. 專案願景與目標

- 願景：成為台灣社區鄰里之間最被信任的工具共享平台。
- 首年目標：累積下載 10,000、月活躍用戶 3,000、每月成功借用 500 筆。

### 2. MVP 範圍

- 第一版納入工具刊登、附近搜尋、借用流程、押金金流、評價等核心功能。
- 信用評分機制與平台保險列為第一版目標功能，視開發進度調整。

### 3. 時程規劃（里程碑）

| 里程碑 | 工作內容 | 期間 |
|---|---|---|
| M1 | 需求確認與 UI/UX 設計 | 2026-01 ~ 2026-02 |
| M2 | MVP 開發 | 2026-03 ~ 2026-05 |
| M3 | 封閉測試（Beta） | 2026-06 ~ 2026-07 |
| M4 | 正式上線 | 2026-08 |

### 4. 預算

專案總預算核定為新台幣 480 萬元，分配如下：

| 項目 | 金額（萬元） |
|---|---|
| 人事費 | 320 |
| 雲端與基礎設施 | 40 |
| 行銷推廣 | 80 |
| 其他與雜支 | 40 |
| 合計 | 480 |
